\subsection{Detailed Description}
This class implements a z-\/domain filter on a specified array of float32's (a.\+k.\+a. singles, floats). It requires the transfer function coefficients be already calculated (i.\+e., it does not generate the coefficients based on desired filter characteristics). M\+A\+T\+L\+A\+B and its Signal Processing Toolbox can be used to generate the coefficients.

While this class is designed to implement a single-\/section filter, several instances of the class can be created and run over the data array in sequence to effectively implement a multi-\/section filter.
